agg - Launch a Zellij session with a vertical tab sidebar.

USAGE:
    agg [SESSION_NAME]
    agg install [--debug]
    agg git-stat PATH...
    agg sessions

ARGS:
    SESSION_NAME    Session name (default: "agg")

SUBCOMMANDS:
    install         Build and deploy the WASM plugin, provision Claude, Cursor,
                    Codex, Gemini and Opencode hooks, and install agg.kdl.
                    Pass --debug for an
                    unoptimised build.
                    Codex hooks are JSON in ~/.codex/hooks.json; you must also enable
                    them in ~/.codex/config.toml: [features] codex_hooks = true
                    (hooks do not run on Windows Codex builds yet).
                    AGG installs Codex `PermissionRequest` hooks for approval waits.
                    Other mid-turn choice/clarification prompts may still keep AGG
                    green until Codex emits Stop.
                    Gemini hooks are JSON in ~/.gemini/settings.json; agg install also
                    enables the Gemini hook system in hooksConfig.
    git-stat        Print "path insertions deletions untracked worktree"
                    per PATH (one line each, in argument order).
    sessions        Show Claude, Codex, and Cursor resumable sessions in a
                    picker and launch the selected one.

Run `agg install` once first, or use `evoke` which runs it for you.
