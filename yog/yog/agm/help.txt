agm - Launch a Zellij session with a vertical tab sidebar.

USAGE:
    agm [SESSION_NAME]
    agm install [--debug]
    agm git-stat PATH...

ARGS:
    SESSION_NAME    Session name (default: "agm")

SUBCOMMANDS:
    install         Build and deploy the WASM plugin, provision Claude, Cursor,
                    Codex, and Opencode hooks. Pass --debug for an unoptimised build.
                    Codex hooks are JSON in ~/.codex/hooks.json; you must also enable
                    them in ~/.codex/config.toml: [features] codex_hooks = true
                    (hooks do not run on Windows Codex builds yet).
    git-stat        Print "path insertions deletions untracked worktree"
                    per PATH (one line each, in argument order).

Requires evoke to be run first (provisions plugin + layout).
