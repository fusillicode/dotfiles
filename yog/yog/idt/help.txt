Usage: idt <dev_tools_dir> <bin_dir> [tool_names...]

Install language servers, linters, formatters, and developer helpers concurrently.

Arguments:
  <dev_tools_dir>  Directory for tool installation.
  <bin_dir>        Directory for binary symlinks.
  [tool_names...]  Optional specific tools to install (default: all).

Examples:
  idt ~/.dev/tools ~/.local/bin              Install all tools.
  idt ~/.dev/tools ~/.local/bin ruff_lsp rust_analyzer  Subset.
