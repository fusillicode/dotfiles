Usage: agg [install [--debug] | git-stat <tab-bar|picker> <paths...> | nudge <summary> <body> <tab-id> <pane-id> [image-path] [--session <session>]]

Zellij AGG plugin helper.

Modes:
  agg install [--debug]              Build and install AGG plugin, layout, hooks, and local assets.
  agg git-stat tab-bar <paths...>    Print dirty status for tab-bar paths. Used by AGG.
  agg git-stat picker <paths...>     Print branch, status, and HEAD commit info for picker paths. Used by AGG.
  agg nudge <summary> <body> <tab-id> <pane-id> [image-path] [--session <session>]
                                     Send a desktop notification. Used by AGG.
