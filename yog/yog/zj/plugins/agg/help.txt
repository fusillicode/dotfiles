Usage: agg [install [--debug] | git-stat <tbar|ppick> <paths...> | nudge <summary> <body> <tab-id> <pane-id> [image-path] [--session <session>]]

Zellij AGG plugin helper.

Modes:
  agg install [--debug]              Build and install AGG plugin, layout, hooks, and local assets.
  agg git-stat tbar <paths...>     Print dirty status for tbar paths. Used by AGG.
  agg git-stat ppick <paths...>    Print branch, status, and HEAD commit info for ppick paths. Used by AGG.
  agg nudge <summary> <body> <tab-id> <pane-id> [image-path] [--session <session>]
                                     Send a desktop notification. Used by AGG.
